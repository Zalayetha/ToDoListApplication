# JackCorp TodoList


<h2>install with git clone</h2>

```
git clone https://github.com/Zalayetha/ToDoListApplication
```

<h3>Features</h3>
<hr>
<ul>
  <li>Add Task</li>
  <li>Delete Task (Swipe to Left)</li>
  <li>Edit Task (Swipe to Right)</li>
  <li>Mark the task as complete (Touch Task)</li>
</ul>
